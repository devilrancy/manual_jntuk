AIM

Write recursive C program for calculation of Factorial of an integer.

PROGRAM
#include<stdio.h>
main()
{ int a, fact;
printf("\nEnter any number: "); scanf ("%d", &a);
fact=rec (a);
printf("\nFactorial Value = %d", fact);
}
rec (int x)
{ int f;
if (x==1) return (1); else f=x*rec(x-1);
return (f);
}
 
OUTPUT
Enter any number: 5
Factorial Value = 120
